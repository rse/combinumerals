FontSite CombiNumerals Ltd

Welcome to Ralf S. Engelschall's small kit for using Sean Cavanaugh's awesome font CombiNumerals Ltd from FontSite on the web to conveniently render circled numbers (in the range 0-99) with the help of a CSS content generation approach. This is especially useful to annotate lines in code blocks and then reference those lines from within the description text.

Render manually:
<i class="cn-raw">4@</>

Render regular circled number N with content-generation:
<i class="cn-N"/>

Render inverse circled number N with content-generation:
<i class="cn-N-i"/>
